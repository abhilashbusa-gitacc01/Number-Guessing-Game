# Number-Guessing-Game
A Number Guessing Game, where system select a number and the user has to guess the correct number,
Guess the number in 3 or less than 3 Attempts,
Beat your best every time.

#----How to Use:
->Copy code Or Download File.
->Goto VS code or other coding platforms.
->Input this file.
->Compile and Run.
